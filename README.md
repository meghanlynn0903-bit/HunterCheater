# 2048 Move Helper

A small React/Vite app that helps choose moves for a 2048-style game where tiles merge by level:

- 1 + 1 = 2
- 2 + 2 = 3
- 3 + 3 = 4

Click a tile to cycle it from blank up to 10. Right-click a tile to clear it.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Upload these files to a GitHub repository.
2. Go to https://vercel.com and sign in with GitHub.
3. Click **Add New Project**.
4. Import your repository.
5. Keep the default settings.
6. Click **Deploy**.
